# מנורת התרעה - פיקוד העורף

מכשיר קטן ושקט שמציג את מצב ההתרעה של פיקוד העורף בעיר שלך באמצעות נורית צבעונית. מיועד להתקנה בממ"ד.

**אדום** -- הישארו בפנים. **ירוק** -- ניתן לצאת.

> **חשוב:** זהו אינו מכשיר מציל חיים. מידע ההתרעה עלול להתעכב בעד שתי
> דקות ביחס לצפירה הרשמית. יש לפעול תמיד לפי הנחיות פיקוד העורף
> הרשמיות. מכשיר זה מיועד כאינדיקטור פסיבי משלים בלבד -- לא כתחליף
> לצפירות או לאפליקציית ההתרעה הרשמית.

## למה זה טוב

- **שקט לחלוטין** -- בלי צלילים, בלי רטט. רק אור.
- **מתאים לשבת** -- פועל ברציפות ללא צורך במגע.
- **אינדיקטור לממ"ד** -- כשזה אדום, ההתרעה עדיין פעילה. כשזה ירוק, ניתנה הרגעה רשמית.
- **בלי לחץ** -- לא צריך לבדוק טלפון או חדשות. מבט בנורית מספיק.

## מצבי הנורית

| נורית | משמעות |
|-------|---------|
| **אדום (קבוע)** | התרעה פעילה בעיר שלך -- הישארו בממ"ד |
| **צהוב (קבוע)** | התרעה מוקדמת -- צפויות התרעות באזור |
| **ירוק (קבוע)** | הכל תקין -- אין התרעות פעילות |
| **כחול (קבוע)** | בעיית תקשורת -- בדקו את האינטרנט |
| **כחול (מהבהב)** | מצב הגדרה -- חברו את הטלפון להגדרה |
| **ירוק (מהבהב)** | עדכון תוכנה בתהליך |

## התחלה מהירה

1. חברו את המכשיר למטען USB-C רגיל (מטען טלפון)
2. בטלפון, התחברו לרשת WiFi בשם **"PikudAlert-Setup"**
3. בחרו את רשת ה-WiFi הביתית, הקלידו סיסמה, בחרו את העיר שלכם
4. הניחו את המכשיר בממ"ד -- זהו

ראו [מדריך ההתקנה המלא](SETUP.he.md) להוראות מפורטות.

## איך זה עובד

המכשיר מתחבר ל-WiFi הביתי ובודק התרעות כל 30 שניות. כשמופעלת התרעה בעיר שלכם, הנורית הופכת לאדום. כשניתנת הרגעה, היא חוזרת לירוק. המידע מגיע מהזנת פיקוד העורף הרשמית דרך שירות ענן מאובטח.

עדכוני תוכנה מתבצעים אוטומטית -- המכשיר בודק גרסאות חדשות מעת לעת ומתעדכן בעצמו. אין צורך בשום פעולה.

## הצהרה חשובה

מכשיר זה מסתמך על חיבור אינטרנט ושירות ענן לקבלת נתוני פיקוד העורף.
מידע ההתרעה עלול להתעכב בעד שתי דקות. המכשיר עלול להפסיק לפעול עקב
הפסקות חשמל, תקלות אינטרנט, או שיבושי שירות.

**אין להסתמך על מכשיר זה כמקור יחיד למידע התרעות.**
הקפידו תמיד על אמצעי התרעה נוסף (אפליקציית פיקוד העורף, רדיו, צפירה).
